# Integrations

Zeno can be used in combination with other libraries, such as benchmarking suites and model chaining frameworks.

| Example                                        | Description                                                                                                                                                                                       |
| ---------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [EleutherAI LM Evaluation Harness](./eleuther) | The [LM Evaluation Harness](https://github.com/EleutherAI/lm-evaluation-harness) by EleutherAI is a popular benchmarking framework for LLMs. Zeno can be used to visualize the benchmark outputs! |
| [Ragas](./ragas)                               | [Ragas](https://docs.ragas.io/en/latest/index.html) is a powerful library for evaluating RAG applications. Dive deeper into the metrics and evaluate your RAG applications with Zeno.             |
