# Integrations

Zeno can be used in combination with other libraries, such as benchmarking suites and model chaining frameworks.
We currently support the following integrations:

| Example                          | Description                                                                                                                                                                                  |
| -------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [EleutherAI Harness](./eleuther) | The [EleutherAI lm evaluation harness](https://github.com/EleutherAI/lm-evaluation-harness) is a benchmarking frmawork for language models. Zeno can be used to visualize benchmark outputs. |
| [Ragas](./ragas)                 | The [Ragas](https://docs.ragas.io/en/latest/index.html) framework can be used to evaluate RAG pipelines. Zeno serves as a visualization framework for these evaluations.                     |
