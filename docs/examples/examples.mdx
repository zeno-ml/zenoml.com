# Use Cases

Our use cases cover different types of data and thus demonstrate various application areas for Zeno.
Currently, we provide the following list of use cases which we continuously expand:

| Example                                | Description                                                                                                                                                                                                                                                     |
| -------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Audio Transcription](./transcription) | Audio transcription is an essential task for applications such as voice assistants, podcast search, and video captioning. In this use case, we compare the performance of the different models on the [Speech Accent Archive](https://accent.gmu.edu/) dataset. |
