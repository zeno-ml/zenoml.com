# Standard Views

Zeno has a number of preconfigured views that can be referenced by name.
The following view names can be passed as the `view` option for a Zeno project.

| View Name                                                                                                           | Description                                                                                                                                                                                 |
| ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [image-classification](https://hub.zenoml.com/playground?params=eyJzYW1wbGUiOiJpbWFnZS1jbGFzc2lmaWNhdGlvbiJ9)       | Display images with ground truth and predicted class labels. Works for both binary and multiclass classification. Requires image inputs and text or numeric outputs.                        |
| [text-classification](https://hub.zenoml.com/playground?params=eyJzYW1wbGUiOiJ0ZXh0LWNsYXNzaWZpY2F0aW9uIn0=)        | Display text with ground truth and predicted class labels. Requires text inputs and text or numeric outputs.                                                                                |
| [audio-transcription](https://hub.zenoml.com/playground?params=eyJzYW1wbGUiOiJhdWRpby10cmFuc2NyaXB0aW9uIn0=)        | Display audio file along with outputed text, e.g. transcription. Requires audio inputs and text outputs.                                                                                    |
| [code-generation](https://hub.zenoml.com/playground?params=eyJzYW1wbGUiOiJjb2RlLWdlbmVyYXRpb24ifQ==)                | Show formatted code input and code predictions. Use for evaluating code generation models such as Codex.                                                                                    |
| [chatbot](https://hub.zenoml.com/playground?params=eyJzYW1wbGUiOiJjaGF0Ym90In0=)                                    | Show input-output pairs from chatbot models using the OpenAI API. See the [API documentation](https://platform.openai.com/docs/guides/chat) for details on the required Chat data format.   |
| [chatbot-markdown](https://hub.zenoml.com/playground?params=eyJzYW1wbGUiOiJjaGF0Ym90LW1hcmtkb3duIn0=)               | Same as `chatbot` but renders each cell block as markdown. Useful for rendering complex interactions with external agents, etc.                                                             |
| [space-separated-values](https://hub.zenoml.com/playground?params=eyJzYW1wbGUiOiJzcGFjZS1zZXBhcmF0ZWQtdmFsdWVzIn0=) | Table view of inputs, outputs, and labels which are space-separated words. Useful for tasks such as part-of-speech tagging.                                                                 |
| [rag](https://hub.zenoml.com/playground?params=eyJzYW1wbGUiOiJyYWcifQ==)                                            | Show the input to a generator that can access information from other documents (retrieval augmented generation). The documents that are used and the system's answer can also be displayed. |
